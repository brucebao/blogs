### Demo
[线上预览地址](http://morepiepie.com)

### Install
```
$ pip install -r requirement.txt
$ python manage.py deploy
$ python manage.py runserver

浏览器打开：127.0.0.1:5000访问
```

### Detail
1. 绝大多数功能需登录才能使用
2. 博客发布，修改，删除
3. 关注和粉丝功能，阅读关注者文章
4. 私信的已读未读
5. 收藏文章和取消收藏
6.  文章分类
7. 热门文章和最新文章显示
8. 注册和邮箱验证
9.  修改密码，邮箱，用户资料
10 分页
